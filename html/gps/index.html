What is the purpose of HTML5 semantic tags? Which ones do you see as most useful?

The use of HTML5 semantic tags helps to better organize the content inside of an HTML document, as well as in providing more enhanced associated meaning to the various elements used in its structure. Using semantic tags to suggest the purpose and role of the content enclosed within an HTML tag improves the automated processing of documents from screen-readers, parsers and search engines. I find tags used for 'sectioning' purposes like the header and footer, nav and section tags the most useful as they help to better organize code by indicating the purpose of each element or code block inside the HTMl doc.

What is the order the browser uses to display elements on a page?

The browser usually displays the elements on a page in the same order they are defined. It parses the HTML structure as a tree so execution happens from top to bottom -the first elements will load first and the last elements will load last. Some browsers for example Firefox vs Chrome may load some particular elements in a different sequence trying to optimize for performance but in general the HTML document gets parsed into a tree and read from top to bottom.
